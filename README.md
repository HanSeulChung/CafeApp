# 카페 메뉴 스마트 주문 서비스 : CafeApp
해당 카페의 메뉴(음식, 음료, MD 굿즈)를 미리 결제하고 주문하는 서비스 입니다.

## 프로젝트 기능 설계
* 회원가입 기능
* 로그인 기능
* 카페 메뉴 장바구니 기능
* 카페 메뉴 결제(주문) 기능
  * 사용자의 위치가 가게의 위치와 반경 10KM이내만 결제(주문) 가능
* 스탬프 적립 기능
  * 결제할 경우 음료 잔수에 맞게 스탬프 적립 기능
  * 스탬프가 10개 모두 적립되었을 경우 쿠폰 1장으로 전환 기능
